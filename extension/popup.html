Privacy & Data Disclaimer

This extension sends the current page URL and/or page text content to a local analysis server for security and AI detection checks.

  • URLs are sent to check for HTTPS, SSL certificates, domain age, phishing indicators, and Google Safe Browsing status.
  • Page text may be extracted and sent for AI-generated content analysis.
  • All processing happens on your local machine — no data is shared with third parties (except Google Safe Browsing API lookups).
